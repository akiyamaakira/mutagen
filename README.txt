perfect dark 的简体中文化

文件：
    language.txt 简体中文语言文件。
    visual.txt   外观字体设置文件。其中设置的宋体貌似对UI无任何影响，请自行取用

适用版本：
perfect dark version 1.15

用法：
设置->其它->导出文件，导出了mutagen文件夹之后，
再把 language.txt 文件复制到该文件夹下替换同名文件，最后重启perfect dark。

P.S.
请确保 language.txt 的编码为 UTF-16LE (带签名)，软件才正常读取，
记事本->另存为->编码：Unicode->保存

visual.txt中设置的宋体貌似对UI无任何影响，请自行取用
