perfect dark version 1.23 简体中文化

------------------------------------

用法:

  1. 关闭 perfect dark
  2. 如果 perfect dark 目录下存在 mutagen 文件夹，先删除 mutagen 文件夹
  3. 启动 perfect dark，设置->其它->导出文件，得到 mutagen 文件夹
  4. 把 language.txt 文件复制到 mutagen 文件夹下替换同名文件
