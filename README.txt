perfect dark version 1.19 简体中文化

-------------------------------

文件:

    language.txt  PD简体中文语言文件。

用法:

  1. 如果 perfect dark 目录下存在 mutagen 文件夹，先删除 mutagen 文件夹
  2. 启动 perfect dark，设置->其它->导出文件，得到 mutagen 文件夹
  3. 把 language.txt 文件复制到 mutagen 文件夹下替换同名文件
  4. 关闭并重新启动 perfect dark

注意事项:

  请确保 language.txt 的编码为 UTF-16LE (带签名)，软件才正常读取，
  方法：记事本打开 language.txt->另存为->编码: Unicode->保存。
