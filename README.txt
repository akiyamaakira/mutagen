perfect dark 的简体中文语言文件

适用版本：
perfect dark version 1.15

用法：
设置->其它->导出文件，导出了mutagen文件夹之后，
再把language.txt文件复制到该文件夹下替换同名文件，最后重启perfect dark。

P.S.
请确保 language.txt 的编码为 UTF-16LE (带签名)，软件才正常读取，
本仓库中的 language.txt 一般都如此，
Windows 默认的记事本->另存为->编码：Unicode->保存
